路由器设置端口映射

服务器的默认端口是22，本地电脑是在路由器后面的，我们需要在路由器设置端口映射将本地电脑的22端口映射出去。我们使用的路由器是TP-Link，登录到服务器的管理页面，在应用->虚拟服务器 里面点击添加即可。

测试

这时候可以通过外网的计算机使用ssh通过22号端口登录到本地计算机。如果连接不上要考虑上面的设置是否正确以及计算机防火墙是否开启端口。

服务器建立ssh隧道并且指定动态转发
