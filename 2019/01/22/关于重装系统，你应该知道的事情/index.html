为什么要重装系统

在决定重装系统之前，大家一定是有充足的理由重装的，我分析了一下，主要应该有以下几点：

  1. 电脑太卡；
  2. 电脑抽风了(出问题)；
  3. 电脑貌似中毒了；
  4. 某个软件装不上；
  5. 我更喜欢某个版本的系统；
  6. C盘满了，得释放点空间或者重新分区；

上面列举的某些情况其实是不用重装的，因为就算重装了，如果你的一些使用习惯不改，过不了多久你依然需要重装系统，我会在以后的文章里面就“电脑为什么这么卡”，“电脑怎样防病毒”阐述一点自己的建议！

重装系统会改变什么

关于重装系统后电脑什么会改变，我相信还是有很多人不知道的，趁这个机会给大家讲讲！

首先，你的原系统里面安装的所有程序都会不见。这点是我们重装系统必须要考虑的时间成本问题，因为你需要将你需要的程序重新安装一遍，其中还包括各种驱动程序。当然如果你是因为电脑中病毒而重装的系统，那么你的新系统里面一般来说是不会有病毒，这里为什么会说一般呢？其一因为前些年出来一款名字叫做“鬼影”的病毒，这个病毒对于一般的重装系统是没有用的！其二是因为我们的电脑有C盘和D盘，重装系统如果没有格式化非系统盘，那么你非系统盘里面的文件如果以前被感染了病毒现在依然是被感染的，一旦你打开里面被感染的文件，你的电脑会再次中毒，所以重要文件一定要定期备份！关于病毒的事情以后的文章里面详细说 : )

其次，你的电脑运行速度可能会加快不少。因为以前可能由于使用问题，电脑的开机启动项特别多，盛放了各种全家桶，一开机内存就被消耗掉了70%。大部分人都喜欢使用免费的东西，但是世界上却没有那么多公益项目，所以用户不付费那么谁来付费呢？广告商，流氓推广！因此我们有时候仅仅是想装个A软件，但是安装完后尽然发现电脑里面莫名其妙出现了B、C、D、E、F等等软件，奇怪的是还不让我们卸载，电脑还时不时得弹出广告窗口，我们还不知道是哪个软件弹出来的！这点程序安装的注意事项我会在“电脑为什么这么卡”一问中详细说明: )

第3点，可能你电脑以前存在的问题会消失。因为毕竟很多时候电脑出现的问题都是软件故障，有些还是软件冲突造成的。但是对于硬件上的故障依然会存在。如果重装系统后发现电脑的问题依然存在，那么就可能是硬件故障了，这种情况就得送修: (

最后，关于电脑里面的文件，我感觉得单独拿出来说一下。windows电脑一般是有分区的。当我们打开文件管理器的时候会看到名字如“C D E F”，那是系统的分区，一般我们的系统都是安装在C盘，这种情况下重装系统只有C盘的文件会改变，其它数据盘的文件是不会改变的(除开自己手动格式化)。因此这里如果你想要节约空间而又不格式化这些盘，可以在重装系统前先把安装在非系统盘的程序给卸载掉（后面删除也是一样的，前提是你得知道什么可以删什么不可以删）。

系统选择的问题

Windows 10出来后，微软就宣布不会再出新的系统了(这个决定以后会不会改谁也不知道，现阶段是这样说的)，也就是说Windows 10是最后一个Windows版本的系统，后面的工作都是对Windows 10的改进和维护。所以现阶段主要供我们选择的系统就是Win7、Win8、Win10。那么当我们决定重新安装系统的时候应该选择哪个版本呢？

首先我得说一下Win8系统。Win8系统出来没多久微软就说要发布Win10,而且Win8用户可以在指定时间免费升级到Win10，为什么？因为Win8非常得失败，出来后大家都不愿意用，都守着Win7不放。看着Win8很失败，微软就加紧了Win10的开发和发布。所以说对于重装系统的我们来说，Win8能不用就不用了。我曾经也用过Win8，时不时得就要出问题，后来Win10出来后赶紧就升级Win10了。说了这么多，就是要提醒各位Win8是很失败的一个版本。

然后再来说说Win7和Win10。Win7现在微软已经不卖了，但是会提供系统更新和维护(想当年微软宣布不再维护和更新XP，国内各大杀毒软件厂商宣称会保护XP用户的计算机的时候自己就想笑，说得他们自己就像比微软都懂Windows系统一样)。对于习惯了使用Win7的用户而又不想要花费时间去折腾和熟悉新系统的人来说，还是选择Win7比较好。如果你现在还在使用Win8，我建议还是使用Win10为好的，毕竟Win8和Win10在很多使用设计上是很相似的。
